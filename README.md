# 🌦️ Weather App

A modern Android Weather Application that displays real-time weather information for any city using the OpenWeather API. The app features a clean user interface, a splash screen with Lottie animations, and responsive XML layouts.

---
##  Features

- Search weather by city name
-  Real-time temperature display
-  Humidity information
- Wind speed details
- Weather condition updates
- Splash screen with Lottie animation
- Responsive Material UI using XML

##  Setup Instructions

### 1. Clone the Repository

```bash
git clone https://github.com/your-username/weather-app.git
```

### 2. Open in Android Studio

- Open Android Studio
- Select **Open**
- Choose the cloned project folder
- Wait for Gradle Sync to complete

### 3. Add Your API Key

Create or update your API key in the appropriate file (e.g., `ApiClient` or `Constants`) using your OpenWeather API key.

### 4. Run the Application

- Connect an Android device or start an emulator
- Click **Run **
